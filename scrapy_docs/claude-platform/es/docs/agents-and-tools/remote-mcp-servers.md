# Servidores MCP remotos

Servidores MCP remotos

---

Varias empresas han desplegado servidores MCP remotos a los que los desarrolladores pueden conectarse a través de la API del conector MCP de Anthropic. Estos servidores amplían las capacidades disponibles para desarrolladores y usuarios finales proporcionando acceso remoto a varios servicios y herramientas a través del protocolo MCP.

<Note>
    Los servidores MCP remotos listados a continuación son servicios de terceros diseñados para funcionar con la API de Claude. Estos servidores
    no son propiedad de, operados por, o respaldados por Anthropic. Los usuarios solo deben conectarse a servidores MCP remotos en los que confíen y
    deben revisar las prácticas de seguridad y términos de cada servidor antes de conectarse.
</Note>

## Conectándose a servidores MCP remotos

Para conectarse a un servidor MCP remoto:

1. Revise la documentación del servidor específico que desea usar.
2. Asegúrese de tener las credenciales de autenticación necesarias.
3. Siga las instrucciones de conexión específicas del servidor proporcionadas por cada empresa.

Para más información sobre el uso de servidores MCP remotos con la API de Claude, consulte la [documentación del conector MCP](/docs/es/agents-and-tools/mcp-connector).
