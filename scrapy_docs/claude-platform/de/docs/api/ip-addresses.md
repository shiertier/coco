# IP-Adressen

Anthropic-Dienste verwenden feste IP-Adressen für eingehende und ausgehende Verbindungen. Sie können diese Adressen verwenden, um Ihre Firewall-Regeln für den sicheren Zugriff auf die Claude API und Console zu konfigurieren. Diese Adressen werden sich ohne Ankündigung nicht ändern.

---

## Eingehende IP-Adressen

Dies sind die IP-Adressen, unter denen Anthropic-Dienste eingehende Verbindungen empfangen.

#### IPv4

`160.79.104.0/23`

#### IPv6

`2607:6bc0::/48`

Dies sind die stabilen IP-Adressen, die Anthropic für ausgehende Anfragen verwendet (zum Beispiel beim Tätigen von MCP-Tool-Aufrufen an externe Server).
