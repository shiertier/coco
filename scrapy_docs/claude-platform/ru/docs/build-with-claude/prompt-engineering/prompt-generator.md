# Автоматическая генерация шаблонов промптов первого черновика

---

<Note>
Наш генератор промптов совместим со всеми моделями Claude, включая те, которые обладают расширенными возможностями мышления. Советы по составлению промптов для моделей с расширенным мышлением смотрите [здесь](/docs/ru/build-with-claude/extended-thinking).
</Note>

Иногда самая сложная часть использования ИИ-модели — это понять, как эффективно составить промпт. Чтобы помочь с этим, мы создали инструмент генерации промптов, который направляет Claude на создание высококачественных шаблонов промптов, адаптированных под ваши конкретные задачи. Эти шаблоны следуют многим нашим лучшим практикам по составлению промптов.

Генератор промптов особенно полезен как инструмент для решения "проблемы чистого листа", давая вам отправную точку для дальнейшего тестирования и итераций.

<Tip>Попробуйте генератор промптов прямо сейчас в [Console](/dashboard).</Tip>

Если вам интересно проанализировать базовый промпт и архитектуру, ознакомьтесь с нашим [блокнотом Google Colab для генератора промптов](https://anthropic.com/metaprompt-notebook/). Там вы можете легко запустить код, чтобы Claude создавал промпты от вашего имени.

<Note>Обратите внимание, что для запуска блокнота Colab вам потребуется [API ключ](/settings/keys).</Note>
