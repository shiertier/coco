# Удаленные MCP серверы

Удаленные MCP серверы

---

Несколько компаний развернули удаленные MCP серверы, к которым разработчики могут подключаться через API коннектора MCP от Anthropic. Эти серверы расширяют возможности, доступные разработчикам и конечным пользователям, предоставляя удаленный доступ к различным сервисам и инструментам через протокол MCP.

<Note>
    Удаленные MCP серверы, перечисленные ниже, являются сторонними сервисами, предназначенными для работы с API Claude. Эти серверы
    не принадлежат, не управляются и не одобряются Anthropic. Пользователи должны подключаться только к тем удаленным MCP серверам, которым они доверяют, и
    должны изучить практики безопасности и условия каждого сервера перед подключением.
</Note>

## Подключение к удаленным MCP серверам

Чтобы подключиться к удаленному MCP серверу:

1. Изучите документацию для конкретного сервера, который вы хотите использовать.
2. Убедитесь, что у вас есть необходимые учетные данные для аутентификации.
3. Следуйте инструкциям по подключению для конкретного сервера, предоставленным каждой компанией.

Для получения дополнительной информации об использовании удаленных MCP серверов с API Claude, см. [документацию по коннектору MCP](/docs/ru/agents-and-tools/mcp-connector).
