# Волшебник электронных таблиц

Создавайте CSV электронные таблицы с различными типами данных.

---

> Скопируйте этот промпт в нашу [Консоль](/dashboard) разработчика, чтобы попробовать самостоятельно!

|        | Содержание |
| --- | --- |
| System | Ваша задача — создать CSV электронную таблицу, содержащую указанный тип данных. Таблица должна быть хорошо организована, с четкими заголовками столбцов и подходящими типами данных для каждого столбца. Убедитесь, что данные реалистичны, разнообразны и отформатированы последовательно. Включите минимум 10 строк данных, не считая строки заголовка. |
| User   | Создайте электронную таблицу, содержащую информацию о книгах в библиотеке. Включите следующие столбцы: ID книги, Название, Автор, Жанр, Год публикации и Доступные экземпляры. Убедитесь, что данные охватывают различные жанры и годы публикации, и что количество доступных экземпляров варьируется между книгами.                             |
