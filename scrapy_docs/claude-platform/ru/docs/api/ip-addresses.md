# IP-адреса

Сервисы Anthropic используют фиксированные IP-адреса для входящих и исходящих соединений. Вы можете использовать эти адреса для настройки правил брандмауэра для безопасного доступа к Claude API и Console. Эти адреса не будут изменяться без уведомления.

---

## Входящие IP-адреса

Это IP-адреса, на которых сервисы Anthropic получают входящие соединения.

#### IPv4

`160.79.104.0/23`

#### IPv6

`2607:6bc0::/48`

## Исходящие IP-адреса

Это стабильные IP-адреса, которые Anthropic использует для исходящих запросов (например, при выполнении вызовов инструментов MCP на внешние серверы).

#### IPv4

`160.79.104.0/21`
