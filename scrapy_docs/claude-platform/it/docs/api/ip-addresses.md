# Indirizzi IP

I servizi Anthropic utilizzano indirizzi IP fissi per le connessioni in entrata e in uscita. Puoi utilizzare questi indirizzi per configurare le regole del firewall per l'accesso sicuro all'API Claude e alla Console. Questi indirizzi non cambieranno senza preavviso.
