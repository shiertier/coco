# Genera automaticamente modelli di prompt di prima bozza

---

<Note>
Il nostro generatore di prompt è compatibile con tutti i modelli Claude, inclusi quelli con capacità di pensiero esteso. Per suggerimenti specifici sui prompt per i modelli con pensiero esteso, consulta [qui](/docs/it/build-with-claude/extended-thinking).
</Note>

A volte, la parte più difficile nell'utilizzo di un modello di IA è capire come sollecitarlo efficacemente. Per aiutare in questo, abbiamo creato uno strumento di generazione di prompt che guida Claude a generare modelli di prompt di alta qualità adattati ai tuoi compiti specifici. Questi modelli seguono molte delle nostre migliori pratiche di ingegneria dei prompt.

Il generatore di prompt è particolarmente utile come strumento per risolvere il "problema della pagina bianca" e darti un punto di partenza per ulteriori test e iterazioni.

<Tip>Prova subito il generatore di prompt direttamente sulla [Console](/dashboard).</Tip>

Se sei interessato ad analizzare il prompt sottostante e l'architettura, dai un'occhiata al nostro [notebook Google Colab del generatore di prompt](https://anthropic.com/metaprompt-notebook/). Lì, puoi facilmente eseguire il codice per far costruire i prompt a Claude per tuo conto.

<Note>Nota che per eseguire il notebook Colab, avrai bisogno di una [chiave API](/settings/keys).</Note>
