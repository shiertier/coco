# Générer automatiquement des modèles de prompts en première version

---

<Note>
Notre générateur de prompts est compatible avec tous les modèles Claude, y compris ceux dotés de capacités de réflexion étendue. Pour des conseils de prompting spécifiques aux modèles à réflexion étendue, consultez [ici](/docs/fr/build-with-claude/extended-thinking).
</Note>

Parfois, la partie la plus difficile de l'utilisation d'un modèle d'IA est de déterminer comment le solliciter efficacement. Pour vous aider, nous avons créé un outil de génération de prompts qui guide Claude pour générer des modèles de prompts de haute qualité adaptés à vos tâches spécifiques. Ces modèles suivent plusieurs de nos meilleures pratiques en matière d'ingénierie de prompts.

Le générateur de prompts est particulièrement utile comme outil pour résoudre le "problème de la page blanche" et vous donner un point de départ pour des tests et des itérations ultérieurs.

<Tip>Essayez le générateur de prompts maintenant directement sur la [Console](/dashboard).</Tip>

Si vous souhaitez analyser le prompt sous-jacent et l'architecture, consultez notre [carnet Google Colab du générateur de prompts](https://anthropic.com/metaprompt-notebook/). Là, vous pourrez facilement exécuter le code pour que Claude construise des prompts en votre nom.

<Note>Notez que pour exécuter le carnet Colab, vous aurez besoin d'une [clé API](/settings/keys).</Note>
