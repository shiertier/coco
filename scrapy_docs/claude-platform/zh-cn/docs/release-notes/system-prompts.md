Claude 的网页界面（[Claude.ai](https://www.claude.ai)）和移动应用使用系统提示词在每次对话开始时向 Claude 提供最新信息，例如当前日期。我们还使用系统提示词来鼓励某些行为，例如始终以 Markdown 格式提供代码片段。随着我们继续改进 Claude 的响应，我们会定期更新此提示词。这些系统提示词更新不适用于 Anthropic API。版本之间的更新以粗体显示。
