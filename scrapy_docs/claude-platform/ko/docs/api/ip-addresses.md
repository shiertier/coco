# IP 주소

Anthropic 서비스는 인바운드 및 아웃바운드 연결 모두에 고정 IP 주소를 사용합니다. 이러한 주소를 사용하여 Claude API 및 Console에 안전하게 액세스하기 위해 방화벽 규칙을 구성할 수 있습니다. 이러한 주소는 사전 통지 없이 변경되지 않습니다.

---

## 인바운드 IP 주소

이는 Anthropic 서비스가 들어오는 연결을 수신하는 IP 주소입니다.

#### IPv4

`160.79.104.0/23`

#### IPv6

`2607:6bc0::/48`

## 아웃바운드 IP 주소

이는 Anthropic이 아웃바운드 요청(예: 외부 서버에 MCP 도구 호출을 할 때)에 사용하는 안정적인 IP 주소입니다.

#### IPv4

`160.79.104.0/21`

*다음의 개별 IP 주소는 여전히 사용 중이지만 2026년 1월 15일부터 단계적으로 폐지될 예정입니다.*

```
34.162.46.92/32
34.162.102.82/32
```
