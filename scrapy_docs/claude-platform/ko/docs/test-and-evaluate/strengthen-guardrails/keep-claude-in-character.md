# 역할 프롬프팅과 사전 채우기로 Claude의 캐릭터 유지하기

---

이 가이드는 긴 복잡한 상호작용 중에도 Claude의 캐릭터를 유지하기 위한 실행 가능한 팁을 제공합니다.

- **시스템 프롬프트로 역할 설정하기:** [시스템 프롬프트](/docs/ko/build-with-claude/prompt-engineering/system-prompts)를 사용하여 Claude의 역할과 성격을 정의하세요. 이는 일관된 응답을 위한 강력한 기반을 마련합니다.
    <Tip>캐릭터를 설정할 때, 성격, 배경, 특정 특성이나 독특한 점에 대한 자세한 정보를 제공하세요. 이는 모델이 캐릭터의 특성을 더 잘 모방하고 일반화하는 데 도움이 됩니다.</Tip>
- **사전 채워진 응답으로 강화하기:** 특히 긴 대화에서는 캐릭터 태그로 Claude의 응답을 사전에 채워 역할을 강화하세요.
- **가능한 시나리오에 대해 Claude 준비시키기:** 프롬프트에 일반적인 시나리오와 예상되는 응답 목록을 제공하세요. 이는 Claude가 캐릭터를 벗어나지 않고 다양한 상황을 처리하도록 "훈련"시킵니다.
