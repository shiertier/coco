# 모델 지원 중단

Anthropic 모델의 지원 중단 및 권장 대체 모델에 대한 정보

---

더 안전하고 성능이 우수한 모델을 출시함에 따라 저희는 정기적으로 이전 모델을 중단합니다. Anthropic 모델을 사용하는 애플리케이션은 계속 작동하도록 가끔 업데이트가 필요할 수 있습니다. 영향을 받는 고객에게는 항상 이메일과 저희 문서를 통해 알림을 드립니다.

이 페이지에는 모든 API 지원 중단 사항과 권장 대체 모델이 나열되어 있습니다.

## 개요

Anthropic은 모델의 생명주기를 설명하기 위해 다음 용어를 사용합니다:
- **활성**: 모델이 완전히 지원되며 사용이 권장됩니다.
- **레거시**: 모델이 더 이상 업데이트를 받지 않으며 향후 지원이 중단될 수 있습니다.
- **지원 중단**: 모델은 신규 고객에게 더 이상 제공되지 않지만 기존 사용자는 중단 날짜까지 계속 사용할 수 있습니다. 이 시점에서 중단 날짜를 지정합니다.
- **중단됨**: 모델을 더 이상 사용할 수 없습니다. 중단된 모델에 대한 요청은 실패합니다.

<Warning>
지원이 중단된 모델은 활성 모델보다 신뢰성이 낮을 가능성이 있습니다. 최고 수준의 지원과 신뢰성을 유지하기 위해 워크로드를 활성 모델로 이동할 것을 강력히 권장합니다.
</Warning>

## 대체 모델로 마이그레이션

모델이 지원 중단되면 중단 날짜 전에 모든 사용을 적절한 대체 모델로 마이그레이션하십시오. 중단 날짜가 지난 모델에 대한 요청은 실패합니다.

대체 모델의 성능을 측정하는 데 도움이 되도록 중단 날짜 훨씬 전에 새 모델로 애플리케이션을 철저히 테스트할 것을 권장합니다.

Claude 3.7에서 Claude 4.5 모델로 마이그레이션하는 방법에 대한 구체적인 지침은 [Claude 4.5로 마이그레이션](/docs/ko/about-claude/models/migrating-to-claude-4)을 참조하십시오.

## 알림

Anthropic은 예정된 중단이 있는 모델의 활성 배포를 사용하는 고객에게 알림을 드립니다. 공개적으로 출시된 모델의 경우 모델 중단 전 최소 60일 전에 알림을 제공합니다.

## API 사용 감사

지원이 중단된 모델의 사용을 파악하기 위해 고객은 API 사용 감사에 액세스할 수 있습니다. 다음 단계를 따르십시오:

1. Console의 [사용](/settings/usage) 페이지로 이동합니다.
2. "내보내기" 버튼을 클릭합니다.
3. 다운로드한 CSV를 검토하여 API 키 및 모델별로 분류된 사용 현황을 확인합니다.

이 감사는 애플리케이션이 여전히 지원이 중단된 모델을 사용하는 인스턴스를 찾는 데 도움이 되므로 중단 날짜 전에 최신 모델로의 업데이트 우선순위를 정할 수 있습니다.

## 모범 사례

1. 모델 지원 중단에 대한 업데이트를 위해 정기적으로 저희 문서를 확인하십시오.
2. 현재 모델의 중단 날짜 훨씬 전에 최신 모델로 애플리케이션을 테스트하십시오.
3. 가능한 한 빨리 권장 대체 모델을 사용하도록 코드를 업데이트하십시오.
4. 마이그레이션에 대한 지원이 필요하거나 질문이 있으면 저희 지원팀에 문의하십시오.

## 지원 중단의 단점 및 완화 방법

저희는 현재 새 모델 출시를 위한 용량을 확보하기 위해 모델을 지원 중단하고 중단합니다. 이에 따른 단점이 있음을 인식합니다:
- 특정 모델을 선호하는 사용자는 새 버전으로 마이그레이션해야 합니다.
- 연구자는 진행 중인 연구 및 비교 연구를 위한 모델에 대한 액세스를 잃습니다.
- 모델 중단은 안전 및 모델 복지 관련 위험을 초래합니다.

언젠가는 과거 모델을 다시 공개적으로 제공할 수 있기를 바랍니다. 그때까지 저희는 모델 가중치의 장기 보존 및 이러한 영향을 완화하기 위한 기타 조치에 약속했습니다. 자세한 내용은 [모델 지원 중단 및 보존에 대한 약속](https://www.anthropic.com/research/deprecation-commitments)을 참조하십시오.
