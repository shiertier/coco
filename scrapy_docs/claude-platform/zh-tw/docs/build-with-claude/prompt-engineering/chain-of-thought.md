# 讓 Claude 思考（思維鏈提示）以提高性能

通過鏈式思維提示技術，讓 Claude 逐步分解問題，提高複雜任務的準確性和連貫性。

---

<Note>
While these tips apply broadly to all Claude models, you can find prompting tips specific to extended thinking models [here](/docs/en/build-with-claude/prompt-engineering/extended-thinking-tips).
</Note>

面對複雜任務（如研究、分析或問題解決）時，給予 Claude 思考的空間可以大幅提高其性能。這種技術被稱為思維鏈（CoT）提示，它鼓勵 Claude 逐步分解問題，從而產生更準確和細緻的輸出。

## 實施 CoT 之前

### 為什麼要讓 Claude 思考？
- **準確性：** 逐步解決問題可以減少錯誤，特別是在數學、邏輯、分析或一般複雜任務中。
- **連貫性：** 結構化思維導致更具凝聚力、組織良好的回應。
- **調試：** 查看 Claude 的思考過程可以幫助您找出提示可能不清楚的地方。

### 為什麼不讓 Claude 思考？
- 增加的輸出長度可能會影響延遲。
- 並非所有任務都需要深入思考。明智地使用 CoT，以確保性能和延遲之間的適當平衡。

<Tip>對於人類需要思考的任務使用 CoT，例如複雜數學、多步驟分析、編寫複雜文檔或涉及多個因素的決策。</Tip>

***

## 如何提示思考

下面的思維鏈技術**按從最簡單到最複雜的順序排列**。較簡單的方法佔用上下文窗口中的空間較少，但通常也不那麼強大。

<Tip>**CoT 提示**：始終讓 Claude 輸出其思考。如果不輸出其思考過程，就不會進行任何思考！</Tip>

- **基本提示**：在您的提示中包含「逐步思考」。
    - 缺乏關於*如何*思考的指導（如果任務非常特定於您的應用、用例或組織，這尤其不理想）
    <section title="示例：編寫捐贈者電子郵件（基本 CoT）">

        | 角色 | 內容 |
        | ---- | ------- |
        | 用戶 | 為捐贈者起草個性化電子郵件，要求他們為今年的「關愛兒童」計劃捐款。<br/><br/>計劃信息：<br/>\\{\{PROGRAM_DETAILS}}<br/>\</program><br/><br/>捐贈者信息：<br/>\<donor>\{\{DONOR_DETAILS}}<br/>\</donor><br/><br/>在編寫電子郵件之前，請逐步思考。 |
    
</section>
- **引導提示**：為 Claude 的思考過程概述具體步驟。
    - 缺乏結構化，使得難以剝離並將答案與思考分開。
    <section title="示例：編寫捐贈者電子郵件（引導 CoT）">

        | 角色 | 內容 |
        | ---- | ------- |
        | 用戶 | 為捐贈者起草個性化電子郵件，要求他們為今年的「關愛兒童」計劃捐款。<br/><br/>計劃信息：<br/>\\{\{PROGRAM_DETAILS}}<br/>\</program><br/><br/>捐贈者信息：<br/>\<donor>\{\{DONOR_DETAILS}}<br/>\</donor><br/><br/>在編寫電子郵件之前進行思考。首先，根據捐贈者的捐款歷史和他們支持過的活動，思考什麼樣的信息可能會吸引這位捐贈者。然後，根據他們的歷史，思考「關愛兒童」計劃的哪些方面會吸引他們。最後，使用您的分析編寫個性化的捐贈者電子郵件。 |
    
</section>
- **結構化提示**：使用 XML 標籤（如 `<thinking>` 和 `<answer>`）將推理與最終答案分開。
     <section title="示例：編寫捐贈者電子郵件（結構化引導 CoT）">

        | 角色 | 內容 |
        | ---- | ------- |
        | 用戶 | 為捐贈者起草個性化電子郵件，要求他們為今年的「關愛兒童」計劃捐款。<br/><br/>計劃信息：<br/>\\{\{PROGRAM_DETAILS}}<br/>\</program><br/><br/>捐贈者信息：<br/>\<donor>\{\{DONOR_DETAILS}}<br/>\</donor><br/><br/>在 \<thinking> 標籤中編寫電子郵件之前進行思考。首先，根據捐贈者的捐款歷史和他們支持過的活動，思考什麼樣的信息可能會吸引這位捐贈者。然後，根據他們的歷史，思考「關愛兒童」計劃的哪些方面會吸引他們。最後，在 \<email> 標籤中編寫個性化的捐贈者電子郵件，使用您的分析。 |
    
</section>
    
### 示例

  <section title="示例：沒有思考的財務分析">

    沒有逐步思考，Claude 給出的建議看起來合理，但缺乏深度。它沒有量化潛在結果或考慮歷史市場表現，這些對於做出明智決策至關重要。
    | 角色 | 內容 |
    | ---- | ------- |
    | 用戶 | 您是一名財務顧問。一位客戶想投資 10,000 美元。他們可以在兩個選項之間選擇：A）一隻歷史上每年回報 12% 但波動較大的股票，或 B）一種保證每年 6% 回報的債券。客戶在 5 年內需要這筆錢作為房屋首付。您會推薦哪個選項？ |
    | 助手 | 根據提供的信息，我會推薦選項 B，即保證 6% 年回報的債券。原因如下：<br/><br/>1. 確定性：債券提供有保證的 6% 年回報，這在財務規劃中提供了確定性。當客戶有特定的財務目標（首付）和固定時間表（5 年）時，這至關重要。<br/>2. 風險厭惡：由於客戶在重要的人生事件（購買房屋）中需要這筆錢，他們可能風險承受能力較低。如果在他們需要資金時股票市場出現下跌，股票市場的波動性可能會危害他們的計劃。<br/>3. 時間範圍：只有 5 年的時間，從潛在的股票市場下跌中恢復的時間較少。更長的投資期限（10 年以上）通常更適合應對股票市場波動。<br/><br/>雖然股票的歷史 12% 回報很吸引人，但過去的表現並不能保證未來的結果。債券的有保證 6% 確保客戶在需要支付首付時會有資金。 |
  
</section>
  <section title="示例：有思考的財務分析">

    通過逐步思考，顧問提供了更豐富的分析。它計算了兩種情景的確切數字，考慮了歷史市場波動性，並深入分析了客戶的風險承受能力。這種徹底的推理導致了更有信心和更有根據的建議。
    | 角色 | 內容 |
    | ---- | ------- |
    | 用戶 | 您是一名財務顧問。一位客戶想投資 10,000 美元。他們可以在兩個選項之間選擇：A）一隻歷史上每年回報 12% 但波動較大的股票，或 B）一種保證每年 6% 回報的債券。客戶在 5 年內需要這筆錢作為房屋首付。您會推薦哪個選項？逐步思考。 |
    | 助手 | 為了確定最適合我們客戶的投資選項，我將逐步考慮幾個因素：<br/><br/>\<thinking><br/>1. 了解客戶的目標和時間表：<br/>   - 目標：房屋首付<br/>   - 時間表：5 年<br/>   - 含義：這是一個短期、高優先級的財務目標。客戶可能無法承受重大風險或波動。<br/><br/>2. 分析選項 A（股票）：<br/>   - 優點：更高的歷史回報（每年 12%）<br/>   - 缺點：波動較大（意味著可能有重大收益或損失）<br/>   - 計算潛在的 5 年回報：<br/>     - 最佳情況（每年 12%）：10,000 美元 * (1.12)^5 ≈ 17,623.42 美元<br/>     - 最壞情況（市場崩潰）：可能損失大部分，例如 50% 崩潰 = 5,000 美元<br/><br/>3. 分析選項 B（債券）：<br/>   - 優點：有保證的回報（每年 6%）<br/>   - 缺點：與歷史股票表現相比回報較低<br/>   - 計算 5 年回報：<br/>     - 有保證的：10,000 美元 * (1.06)^5 ≈ 13,382.25 美元<br/><br/>4. 考慮風險承受能力：<br/>   - 目標是房屋首付（必需，非可選）<br/>   - 短時間表（5 年）限制了從市場下跌中恢復的能力<br/>   - 結論：客戶可能風險承受能力較低<br/><br/>5. 歷史視角：<br/>   - 股票的 12% 是歷史性的，不是有保證的<br/>   - 在 5 年期間，股票市場可能波動很大：<br/>     - 2003-2007（金融危機前）：標普 500 指數上漲約 82%<br/>     - 2007-2011（危機期間）：標普 500 指數下跌約 2%<br/>   - 結論：過去的表現不能保證未來的結果，特別是短期<br/><br/>6. 最終分析：<br/>   - 選項 A（股票）：更高的潛在回報，但無法接受的風險，可能無法達到首付目標<br/>   - 選項 B（債券）：保證達到最佳情況股票情景的約 80%，100% 確定有資金<br/>\</thinking><br/><br/>\<answer><br/>我推薦選項 B，即保證 6% 年回報的債券。雖然股票的歷史 12% 回報很誘人，但債券確保您在 5 年內將有 13,382.25 美元用於房屋首付。鑑於您目標的重要性和短時間表，股票的波動性構成了無法接受的風險。債券提供了確定性，這對於這樣一個關鍵的財務里程碑是無價的。<br/>\</answer> |
  
</section>
