# Endereços IP

Os serviços da Anthropic usam endereços IP fixos para conexões de entrada e saída. Você pode usar esses endereços para configurar suas regras de firewall para acesso seguro à API Claude e Console. Esses endereços não serão alterados sem aviso prévio.

---

## Endereços IP de entrada

Estes são os endereços IP onde os serviços da Anthropic recebem conexões de entrada.

#### IPv4

`160.79.104.0/23`

#### IPv6

`2607:6bc0::/48`

## Endereços IP de saída

Estes são os endereços IP estáveis que a Anthropic usa para solicitações de saída (por exemplo, ao fazer chamadas de ferramentas MCP para servidores externos).
