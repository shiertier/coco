# Gerar automaticamente modelos de prompt de primeiro rascunho

---

<Note>
Nosso gerador de prompts é compatível com todos os modelos Claude, incluindo aqueles com capacidades de pensamento estendido. Para dicas de prompting específicas para modelos de pensamento estendido, veja [aqui](/docs/pt-BR/build-with-claude/extended-thinking).
</Note>

Às vezes, a parte mais difícil de usar um modelo de IA é descobrir como fazer prompts de forma eficaz. Para ajudar com isso, criamos uma ferramenta de geração de prompts que orienta o Claude a gerar modelos de prompts de alta qualidade adaptados às suas tarefas específicas. Esses modelos seguem muitas das nossas melhores práticas de engenharia de prompts.

O gerador de prompts é particularmente útil como uma ferramenta para resolver o "problema da página em branco" e dar a você um ponto de partida para testes e iterações adicionais.

<Tip>Experimente o gerador de prompts agora diretamente no [Console](/dashboard).</Tip>

Se você estiver interessado em analisar o prompt subjacente e a arquitetura, confira nosso [notebook Google Colab do gerador de prompts](https://anthropic.com/metaprompt-notebook/). Lá, você pode facilmente executar o código para que o Claude construa prompts em seu nome.

<Note>Observe que para executar o notebook Colab, você precisará de uma [chave de API](/settings/keys).</Note>
