# Servidores MCP remotos

Servidores MCP remotos

---

Várias empresas implantaram servidores MCP remotos aos quais os desenvolvedores podem se conectar através da API do conector MCP da Anthropic. Esses servidores expandem as capacidades disponíveis para desenvolvedores e usuários finais, fornecendo acesso remoto a vários serviços e ferramentas através do protocolo MCP.

<Note>
    Os servidores MCP remotos listados abaixo são serviços de terceiros projetados para funcionar com a API Claude. Esses servidores
    não são de propriedade, operados ou endossados pela Anthropic. Os usuários devem se conectar apenas a servidores MCP remotos em que confiam e
    devem revisar as práticas de segurança e termos de cada servidor antes de se conectar.
</Note>

## Conectando-se a servidores MCP remotos

Para se conectar a um servidor MCP remoto:

1. Revise a documentação do servidor específico que você deseja usar.
2. Certifique-se de ter as credenciais de autenticação necessárias.
3. Siga as instruções de conexão específicas do servidor fornecidas por cada empresa.

Para mais informações sobre o uso de servidores MCP remotos com a API Claude, consulte a [documentação do conector MCP](/docs/pt-BR/agents-and-tools/mcp-connector).
