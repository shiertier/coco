# 遠端 MCP 伺服器

遠端 MCP 伺服器

---

數家公司已部署遠端 MCP 伺服器，開發者可透過 Anthropic MCP 連接器 API 連接到這些伺服器。這些伺服器透過 MCP 協定提供對各種服務和工具的遠端存取，擴展了開發者和終端使用者可用的功能。

<Note>
    下列遠端 MCP 伺服器是設計與 Claude API 搭配使用的第三方服務。這些伺服器
    並非由 Anthropic 擁有、營運或認可。使用者應僅連接到他們信任的遠端 MCP 伺服器，
    並應在連接前檢視每個伺服器的安全實務和條款。
</Note>

## 連接到遠端 MCP 伺服器

要連接到遠端 MCP 伺服器：

1. 檢視您想要使用的特定伺服器文件。
2. 確保您擁有必要的身份驗證憑證。
3. 遵循每家公司提供的伺服器特定連接說明。

有關使用遠端 MCP 伺服器與 Claude API 的更多資訊，請參閱 [MCP 連接器文件](/docs/zh-TW/agents-and-tools/mcp-connector)。
