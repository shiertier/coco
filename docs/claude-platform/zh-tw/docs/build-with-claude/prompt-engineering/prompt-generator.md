# 自動生成初稿提示詞模板

使用我們的提示詞生成工具，為您的特定任務生成高品質的提示詞模板。

---

<Note>
我們的提示詞生成器與所有 Claude 模型相容，包括具有延伸思考能力的模型。關於延伸思考模型的特定提示技巧，請參見[這裡](/docs/zh-TW/build-with-claude/extended-thinking)。
</Note>

有時候，使用 AI 模型最困難的部分就是弄清楚如何有效地提示它。為了解決這個問題，我們創建了一個提示詞生成工具，引導 Claude 生成針對您特定任務的高品質提示詞模板。這些模板遵循了我們許多提示工程的最佳實踐。

提示詞生成器特別適合用來解決「空白頁面問題」，為您提供進一步測試和迭代的起點。

<Tip>現在就在 [Console](/dashboard) 上試用提示詞生成器。</Tip>

如果您對分析底層提示詞和架構感興趣，請查看我們的 [提示詞生成器 Google Colab 筆記本](https://anthropic.com/metaprompt-notebook/)。在那裡，您可以輕鬆運行代碼，讓 Claude 為您構建提示詞。

<Note>請注意，要運行 Colab 筆記本，您需要一個 [API 金鑰](/settings/keys)。</Note>
