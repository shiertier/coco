# Automatische Generierung von Prompt-Vorlagen im ersten Entwurf

Ein Tool zur automatischen Generierung von Prompt-Vorlagen für Claude

---

<Note>
Unser Prompt-Generator ist mit allen Claude-Modellen kompatibel, einschließlich derer mit erweiterten Denkfähigkeiten. Tipps zum Prompting für Modelle mit erweiterten Denkfähigkeiten finden Sie [hier](/docs/de/build-with-claude/extended-thinking).
</Note>

Manchmal ist der schwierigste Teil bei der Nutzung eines KI-Modells herauszufinden, wie man es effektiv anweist. Um dabei zu helfen, haben wir ein Prompt-Generierungstool entwickelt, das Claude dabei unterstützt, hochwertige Prompt-Vorlagen zu erstellen, die auf Ihre spezifischen Aufgaben zugeschnitten sind. Diese Vorlagen folgen vielen unserer Best Practices für Prompt-Engineering.

Der Prompt-Generator ist besonders nützlich als Werkzeug zur Lösung des "leere Seite Problems" und gibt Ihnen einen Ausgangspunkt für weitere Tests und Iterationen.

<Tip>Probieren Sie den Prompt-Generator jetzt direkt in der [Console](/dashboard) aus.</Tip>

Wenn Sie sich für die Analyse des zugrundeliegenden Prompts und der Architektur interessieren, schauen Sie sich unser [Prompt-Generator Google Colab Notebook](https://anthropic.com/metaprompt-notebook/) an. Dort können Sie den Code einfach ausführen, damit Claude Prompts für Sie erstellt.

<Note>Beachten Sie, dass Sie für die Ausführung des Colab Notebooks einen [API-Schlüssel](/settings/keys) benötigen.</Note>
