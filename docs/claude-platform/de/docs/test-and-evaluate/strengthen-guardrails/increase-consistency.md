# Erhöhen Sie die Konsistenz der Ausgabe

Erfahren Sie, wie Sie die Konsistenz von Claudes Antworten durch präzise Formatvorgaben, Prefilling, Beispiele und Retrieval verbessern.

---

<Tip>
**Für garantierte JSON-Schema-Konformität**

Wenn Sie sicherstellen müssen, dass Claude immer gültiges JSON ausgibt, das einem bestimmten Schema entspricht, verwenden Sie stattdessen [Structured Outputs](/docs/de/build-with-claude/structured-outputs). Structured Outputs bieten garantierte Schema-Konformität und sind speziell für diesen Anwendungsfall konzipiert.

Die folgenden Techniken sind nützlich für allgemeine Ausgabekonsistenz oder wenn Sie Flexibilität über strikte JSON-Schemas hinaus benötigen.
</Tip>

So machen Sie Claudes Antworten konsistenter:

## Geben Sie das gewünschte Ausgabeformat an
Definieren Sie Ihr gewünschtes Ausgabeformat präzise mit JSON, XML oder benutzerdefinierten Vorlagen, damit Claude jedes Ausgabeformatierungselement versteht, das Sie benötigen.

<section title="Beispiel: Standardisierung von Kundenfeedback">

    | Rolle | Inhalt |
    | ---- | ------- |
    | Benutzer | Du bist eine Customer Insights AI. Analysiere dieses Feedback und gib es im JSON-Format mit den Schlüsseln aus: "sentiment" (positive/negative/neutral), "key_issues" (Liste) und "action_items" (Liste von Dicts mit "team" und "task").<br/><br/>"Ich bin seit 3 Jahren ein treuer Benutzer, aber das kürzliche UI-Update ist eine Katastrophe. Grundlegende Funktionen zu finden ist jetzt eine Schnitzeljagd. Außerdem ist die neue 'Premium'-Preisgestaltung unverschämt. Ich erwäge einen Wechsel, wenn das nicht sofort behoben wird." |
    | Assistent | \{<br/>  "sentiment": "negative",<br/>  "key_issues": [<br/>    "Schlechte UI/UX im letzten Update",<br/>    "Schwierigkeit, grundlegende Funktionen zu finden",<br/>    "Wahrgenommene hohe Preisgestaltung (neue 'Premium'-Stufe)"<br/>  ],<br/>  "action_items": [<br/>    \{<br/>      "team": "Product",<br/>      "task": "Usability-Tests durchführen und UI basierend auf Erkenntnissen überarbeiten"<br/>    \},<br/>    \{<br/>      "team": "UX",<br/>      "task": "Einen Feature-Discovery-Leitfaden oder Onboarding-Flow erstellen"<br/>    \},<br/>    \{<br/>      "team": "Marketing",<br/>      "task": "Preisgestaltungsstrategie überprüfen und Wertversprechen kommunizieren"<br/>    \},<br/>    \{<br/>      "team": "Customer Success",<br/>      "task": "Benutzer kontaktieren, Zugeständnisse anbieten, um Abwanderung zu verhindern"<br/>    \}<br/>  ]<br/>\} |

</section>

## Füllen Sie Claudes Antwort vor
Füllen Sie den `Assistant`-Turn mit Ihrem gewünschten Format vor. Dieser Trick umgeht Claudes freundliche Einleitung und erzwingt Ihre Struktur.

<section title="Beispiel: Täglicher Verkaufsbericht">

    | Rolle | Inhalt |
    | ---- | ------- |
    | Benutzer | Du bist eine einsichtsvolle Sales Intelligence AI. Erstelle den heutigen Verkaufsbericht.<br/><br/>Strukturiere den Bericht wie folgt:<br/><br/>\<report><br/>    \<summary><br/>        \<metric name="total_revenue">\$0.00\</metric><br/>        \<metric name="units_sold">0\</metric><br/>        \<metric name="avg_order_value">\$0.00\</metric><br/>    \</summary><br/>    \<top_products><br/>        \<br/>            \<name>Product Name\</name><br/>            \<revenue>\$0.00\</revenue><br/>            \<units>0\</units><br/>        \</product><br/>        ...<br/>    \</top_products><br/>    \<regional_performance><br/>        \<region><br/>            \<name>Region Name\</name><br/>            \<revenue>\$0.00\</revenue><br/>            \<growth>0.0%\</growth><br/>        \</region><br/>        ...<br/>    \</regional_performance><br/>    \<action_items><br/>        \<item>Action item.\</item><br/>        ...<br/>    \</action_items><br/>\</report> |
    | Assistent (Prefill) | \<report><br/>    \<summary><br/>        \<metric name= |
    | Assistent | "total_revenue">\$842,567.00\</metric><br/>        \<metric name="units_sold">15,238\</metric><br/>        \<metric name="avg_order_value">\$55.29\</metric><br/>    \</summary><br/>    \<top_products><br/>        \<br/>            \<name>AcmeGuard Pro\</name><br/>            \<revenue>\$325,890.00\</revenue><br/>            \<units>3,621\</units><br/>        \</product><br/>        \<br/>            \<name>AcmeVault Enterprise\</name><br/>            \<revenue>\$254,400.00\</revenue><br/>            \<units>1,272\</units><br/>        \</product><br/>        \<br/>            \<name>AcmeBoost XL\</name><br/>            \<revenue>\$147,690.00\</revenue><br/>            \<units>4,923\</units><br/>        \</product><br/>    \</top_products><br/>    \<regional_performance><br/>        \<region><br/>            \<name>APAC\</name><br/>            \<revenue>\$258,960.00\</revenue><br/>            \<growth>12.8%\</growth><br/>        \</region><br/>        \<region><br/>            \<name>EMEA\</name><br/>            \<revenue>\$325,478.00\</revenue><br/>            \<growth>3.2%\</growth><br/>        \</region><br/>        \<region><br/>            \<name>Americas\</name><br/>            \<revenue>\$258,129.00\</revenue><br/>            \<growth>-2.1%\</growth><br/>        \</region><br/>    \</regional_performance><br/>    \<action_items><br/>        \<item>Untersuchen Sie den Umsatzrückgang in Amerika; planen Sie ein Treffen mit dem regionalen Verkaufsteam.\</item><br/>        \<item>Beschleunigen Sie die CloudGuard Pro-Einführung in APAC, um das Wachstum zu nutzen.\</item><br/>        \<item>Überprüfen Sie die NetBoost XL-Preisgestaltung; hohes Volumen, aber niedrigerer Umsatz.\</item><br/>    \</action_items><br/>\</report> |

</section>

## Mit Beispielen einschränken
Geben Sie Beispiele Ihrer gewünschten Ausgabe an. Dies trainiert Claudes Verständnis besser als abstrakte Anweisungen.

<section title="Beispiel: Konsistente Marktintelligenz generieren">

    | Rolle | Inhalt |
    | ---- | ------- |

Hier sind unsere Konkurrentendaten:<br/>\<data><br/>\{\{DATA}}<br/>\</data><br/><br/>Geben Sie das folgende Beispielformat aus:<br/><br/>\<competitor><br/>  \<name>Rival Inc\</name><br/>  \<overview>Eine 50-Wort-Zusammenfassung.\</overview><br/>  \<swot><br/>    \<strengths>- Aufzählungspunkte\</strengths><br/>    \<weaknesses>- Aufzählungspunkte\</weaknesses><br/>    \<opportunities>- Aufzählungspunkte\</opportunities><br/>    \<threats>- Aufzählungspunkte\</threats><br/>  \</swot><br/>  \<strategy>Eine 30-Wort-strategische Antwort.\</strategy><br/>\</competitor><br/><br/>Analysieren Sie nun AcmeGiant und AcmeDataCo mit diesem Format. |
    | Assistent | \<competitor><br/>  \<name>AcmeGiant\</name><br/>  \<overview>Globaler Tech-Anführer mit einem riesigen Ökosystem von Verbraucher- und Unternehmensprodukten. ## Verwenden Sie Retrieval für kontextuelle Konsistenz
Für Aufgaben, die konsistenten Kontext erfordern (z. B. Chatbots, Wissensdatenbanken), verwenden Sie Retrieval, um Claudes Antworten in einem festen Informationssatz zu verankern.

<section title="Beispiel: Verbesserung der IT-Support-Konsistenz">

    | Rolle | Inhalt |
    | ---- | ------- |
    | Benutzer | Du bist unsere IT-Support-AI, die auf Wissensdatenbankdaten zurückgreift. Hier sind Einträge aus Ihrer Wissensdatenbank:<br/><br/>\<kb><br/>  \<entry><br/>    \<id>1\</id><br/>    \<title>Active Directory-Passwort zurücksetzen\</title><br/>    \<content>1. Gehen Sie zu password.ourcompany.com<br/>2. Geben Sie Ihren Benutzernamen ein<br/>3. Klicken Sie auf "Passwort vergessen"<br/>4. Folgen Sie den E-Mail-Anweisungen\</content><br/>  \</entry><br/>  \<entry><br/>    \<id>2\</id><br/>    \<title>Mit VPN verbinden\</title><br/>    \<content>1. Installieren Sie GlobalProtect aus dem Software Center<br/>2. Öffnen Sie GlobalProtect, geben Sie "vpn.ourcompany.com" ein<br/>3. Verwenden Sie AD-Anmeldedaten zum Anmelden\</content><br/>  \</entry><br/>\</kb><br/><br/>Überprüfen Sie bei der Unterstützung von Benutzern immer zuerst die Wissensdatenbank. Antworten Sie in diesem Format:<br/><br/>\<response><br/>  \<kb_entry>Verwendeter Wissensdatenbankeinträg\</kb_entry><br/>  \<answer>Ihre Antwort\</answer><br/>\</response> Schreiben Sie einige Testfragen für sich selbst und beantworten Sie sie mit der Wissensdatenbank, um sicherzustellen, dass Sie verstehen, wie Sie die Wissensdatenbank richtig verwenden. |
    | Assistent | Verstanden. Ich werde die Wissensdatenbank für konsistente Antworten verwenden. Lassen Sie uns mit ein paar Szenarien testen.<br/><br/>Benutzer: Hey, ich kann mich nicht in meine E-Mail einloggen. Ich denke, ich habe mein Passwort vergessen.<br/><br/>\<response><br/>  \<kb_entry>1: Active Directory-Passwort zurücksetzen\</kb_entry><br/>  \<answer>Ich verstehe, dass Sie Probleme beim Anmelden in Ihre E-Mail haben, weil Sie Ihr Passwort vergessen haben. Lassen Sie mich wissen, wenn Sie weitere Hilfe benötigen.\</answer><br/>\</response><br/><br/>Benutzer: Ich arbeite heute von zu Hause aus. Lassen Sie mich wissen, wenn Sie während der Einrichtung auf Probleme stoßen.\</answer><br/>\</response> |

</section>

## Verketten Sie Prompts für komplexe Aufgaben
Teilen Sie komplexe Aufgaben in kleinere, konsistente Teilaufgaben auf. Jede Teilaufgabe erhält Claudes volle Aufmerksamkeit, wodurch Inkonsistenzfehler in skalierten Workflows reduziert werden.
