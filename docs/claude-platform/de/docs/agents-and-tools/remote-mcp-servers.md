# Remote MCP-Server

Remote MCP-Server

---

Mehrere Unternehmen haben Remote-MCP-Server bereitgestellt, mit denen sich Entwickler über die Anthropic MCP-Connector-API verbinden können. Diese Server erweitern die für Entwickler und Endbenutzer verfügbaren Funktionen, indem sie Remote-Zugriff auf verschiedene Dienste und Tools über das MCP-Protokoll bereitstellen.

<Note>
    Die unten aufgeführten Remote-MCP-Server sind Drittanbieterdienste, die für die Verwendung mit der Claude-API entwickelt wurden. Diese Server
    werden nicht von Anthropic besessen, betrieben oder unterstützt. Benutzer sollten sich nur mit Remote-MCP-Servern verbinden, denen sie vertrauen, und
    sollten die Sicherheitspraktiken und Bedingungen jedes Servers vor der Verbindung überprüfen.
</Note>

## Verbindung zu Remote-MCP-Servern

Um sich mit einem Remote-MCP-Server zu verbinden:

1. Überprüfen Sie die Dokumentation für den spezifischen Server, den Sie verwenden möchten.
2. Stellen Sie sicher, dass Sie über die erforderlichen Authentifizierungsdaten verfügen.
3. Befolgen Sie die serverspezifischen Verbindungsanweisungen, die von jedem Unternehmen bereitgestellt werden.

Weitere Informationen zur Verwendung von Remote-MCP-Servern mit der Claude-API finden Sie in der [MCP-Connector-Dokumentation](/docs/de/agents-and-tools/mcp-connector).
