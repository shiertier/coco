# Invites système

Consultez les mises à jour des invites système principales sur [Claude.ai](https://www.claude.ai) et les applications Claude [iOS](http://anthropic.com/ios) et [Android](http://anthropic.com/android).

---

L'interface web de Claude ([Claude.ai](https://www.claude.ai)) et les applications mobiles utilisent une invite système pour fournir des informations à jour, telles que la date actuelle, à Claude au début de chaque conversation. Nous utilisons également l'invite système pour encourager certains comportements, comme toujours fournir des extraits de code en Markdown. Nous mettons à jour périodiquement cette invite au fur et à mesure que nous continuons à améliorer les réponses de Claude. Ces mises à jour d'invite système ne s'appliquent pas à l'API Anthropic. Les mises à jour entre les versions sont en gras.
