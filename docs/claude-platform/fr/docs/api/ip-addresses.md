# Adresses IP

Les services Anthropic utilisent des adresses IP fixes pour les connexions entrantes et sortantes. Vous pouvez utiliser ces adresses pour configurer vos règles de pare-feu pour un accès sécurisé à l'API Claude et à la Console. Ces adresses ne changeront pas sans préavis.

---

## Adresses IP entrantes

Ce sont les adresses IP où les services Anthropic reçoivent les connexions entrantes.

#### IPv4

`160.79.104.0/23`

#### IPv6

`2607:6bc0::/48`

## Adresses IP sortantes

Ce sont les adresses IP stables qu'Anthropic utilise pour les demandes sortantes (par exemple, lors d'appels d'outils MCP à des serveurs externes).

#### IPv4

`160.79.104.0/21`
