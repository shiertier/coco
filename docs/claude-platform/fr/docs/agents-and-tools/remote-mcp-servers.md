# Serveurs MCP distants

---

Plusieurs entreprises ont déployé des serveurs MCP distants auxquels les développeurs peuvent se connecter via l'API du connecteur MCP d'Anthropic. Ces serveurs étendent les capacités disponibles aux développeurs et aux utilisateurs finaux en fournissant un accès distant à divers services et outils via le protocole MCP.

<Note>
    Les serveurs MCP distants listés ci-dessous sont des services tiers conçus pour fonctionner avec l'API Claude. Ces serveurs
    ne sont pas détenus, exploités ou approuvés par Anthropic. Les utilisateurs ne doivent se connecter qu'aux serveurs MCP distants en lesquels ils ont confiance et
    doivent examiner les pratiques de sécurité et les conditions de chaque serveur avant de se connecter.
</Note>

## Connexion aux serveurs MCP distants

Pour se connecter à un serveur MCP distant :

1. Consultez la documentation du serveur spécifique que vous souhaitez utiliser.
2. Assurez-vous d'avoir les identifiants d'authentification nécessaires.
3. Suivez les instructions de connexion spécifiques au serveur fournies par chaque entreprise.

Pour plus d'informations sur l'utilisation des serveurs MCP distants avec l'API Claude, consultez la [documentation du connecteur MCP](/docs/fr/agents-and-tools/mcp-connector).
