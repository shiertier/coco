# IP 地址

Anthropic 服务对入站和出站连接使用固定的 IP 地址。您可以使用这些地址来配置防火墙规则，以安全地访问 Claude API 和控制台。这些地址不会在没有通知的情况下更改。

---

## 入站 IP 地址

这些是 Anthropic 服务接收传入连接的 IP 地址。

#### IPv4

`160.79.104.0/23`

#### IPv6

`2607:6bc0::/48`
