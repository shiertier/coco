# 自动生成首版提示词模板

---

<Note>
我们的提示词生成器与所有 Claude 模型兼容，包括那些具有扩展思维能力的模型。关于扩展思维模型的具体提示技巧，请参见[这里](/docs/zh-CN/build-with-claude/extended-thinking)。
</Note>

有时候，使用 AI 模型最困难的部分是弄清楚如何有效地进行提示。为了解决这个问题，我们创建了一个提示词生成工具，引导 Claude 生成针对您特定任务的高质量提示词模板。这些模板遵循了我们许多提示工程的最佳实践。

提示词生成器作为解决"空白页问题"的工具特别有用，为您提供进一步测试和迭代的起点。

<Tip>现在就在 [Console](/dashboard) 上试用提示词生成器。</Tip>

如果您对分析底层提示词和架构感兴趣，请查看我们的[提示词生成器 Google Colab 笔记本](https://anthropic.com/metaprompt-notebook/)。在那里，您可以轻松运行代码，让 Claude 为您构建提示词。

<Note>请注意，要运行 Colab 笔记本，您需要一个 [API 密钥](/settings/keys)。</Note>
