# Direcciones IP

Los servicios de Anthropic utilizan direcciones IP fijas para conexiones entrantes y salientes. Puede utilizar estas direcciones para configurar las reglas de su firewall para acceso seguro a la API de Claude y la Consola. Estas direcciones no cambiarán sin previo aviso.

---

## Direcciones IP entrantes

Estas son las direcciones IP donde los servicios de Anthropic reciben conexiones entrantes.

#### IPv4

`160.79.104.0/23`

#### IPv6

`2607:6bc0::/48`

## Direcciones IP salientes

Estas son las direcciones IP estables que Anthropic utiliza para solicitudes salientes (por ejemplo, al realizar llamadas de herramientas MCP a servidores externos).

#### IPv4

`160.79.104.0/21`
