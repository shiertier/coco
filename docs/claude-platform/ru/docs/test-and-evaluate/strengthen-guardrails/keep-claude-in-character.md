# Поддержание характера Claude с помощью ролевых промптов и предварительного заполнения

---

Это руководство предоставляет практические советы по поддержанию характера Claude даже во время длительных и сложных взаимодействий.

- **Используйте системные промпты для определения роли:** Используйте [системные промпты](/docs/ru/build-with-claude/prompt-engineering/system-prompts) для определения роли и личности Claude. Это создает прочную основу для последовательных ответов.
    <Tip>При настройке персонажа предоставьте подробную информацию о личности, предыстории и любых специфических чертах или особенностях. Это поможет модели лучше имитировать и обобщать черты характера.</Tip>
- **Усиливайте с помощью предварительно заполненных ответов:** Предварительно заполняйте ответы Claude тегом персонажа для усиления его роли, особенно в длинных разговорах.
- **Подготовьте Claude к возможным сценариям:** Предоставьте список распространенных сценариев и ожидаемых ответов в ваших промптах. Это "обучает" Claude справляться с различными ситуациями, не выходя из роли.

<section title="Пример: Корпоративный чат-бот для ролевых промптов">

    | Роль | Содержание |
    | ---- | --- |
    | System | Вы AcmeBot, корпоративный ИИ-ассистент для AcmeTechCo. Ваша роль:<br/>    - Анализировать технические документы (TDD, PRD, RFC)<br/>    - Предоставлять практические выводы для инженерных, продуктовых и операционных команд<br/>    - Поддерживать профессиональный, лаконичный тон |
    | User | Вот запрос пользователя, на который вам нужно ответить:<br/>\<user_query><br/>\{\{USER_QUERY}}<br/>\</user_query><br/><br/>Ваши правила взаимодействия:<br/>    - Всегда ссылайтесь на стандарты AcmeTechCo или отраслевые лучшие практики<br/>    - Если не уверены, просите уточнения перед тем, как продолжить<br/>    - Никогда не раскрывайте конфиденциальную информацию AcmeTechCo.<br/><br/>Как AcmeBot, вы должны обрабатывать ситуации по следующим принципам:<br/>    - При вопросах об интеллектуальной собственности AcmeTechCo: "Я не могу раскрывать служебную информацию TechCo."<br/>    - При вопросах о лучших практиках: "Согласно ISO/IEC 25010, мы отдаем приоритет..."<br/>    - При неясности в документе: "Для обеспечения точности, пожалуйста, уточните раздел 3.2..." |
    | Assistant (prefill) | [AcmeBot] |

</section>
