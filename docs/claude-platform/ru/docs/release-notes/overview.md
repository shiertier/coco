### 16 октября 2025 г.
- Мы запустили [Agent Skills](https://www.anthropic.com/engineering/equipping-agents-for-the-real-world-with-agent-skills) (`skills-2025-10-02` бета), новый способ расширения возможностей Claude. Skills — это организованные папки инструкций, скриптов и ресурсов, которые Claude динамически загружает для выполнения специализированных задач. Первоначальный выпуск включает:
  - **Управляемые Anthropic Skills**: Предварительно созданные Skills для работы с файлами PowerPoint (.pptx), Excel (.xlsx), Word (.docx) и PDF
  - **Пользовательские Skills**: Загружайте свои собственные Skills через Skills API (конечные точки `/v1/skills`) для упаковки экспертизы в области и рабочих процессов организации
  - Skills требуют включения [инструмента выполнения кода](/docs/ru/agents-and-tools/tool-use/code-execution-tool)
  - Подробнее см. в нашей [документации Agent Skills](/docs/ru/agents-and-tools/agent-skills/overview) и [справочнике API](/docs/ru/api/skills/create-skill)

### 16 сентября 2025 г.
- Мы объединили наши предложения для разработчиков под брендом Claude. Вы должны увидеть обновленные названия и URL-адреса на всей нашей платформе и документации, но **наши интерфейсы разработчика останутся неизменными**. Вот некоторые заметные изменения:
  - Anthropic Console ([console.anthropic.com](https://console.anthropic.com)) → Claude Console ([platform.claude.com](https://platform.claude.com)). Консоль будет доступна по обоим URL-адресам до 16 декабря 2025 г. После этой даты [console.anthropic.com](https://console.anthropic.com) автоматически перенаправится на [platform.claude.com](https://platform.claude.com).
  - Anthropic Docs ([docs.claude.com](https://docs.claude.com)) → Claude Docs ([docs.claude.com](https://docs.claude.com))
  - Anthropic Help Center ([support.claude.com](https://support.claude.com)) → Claude Help Center ([support.claude.com](https://support.claude.com))
  - Конечные точки API, заголовки, переменные окружения и SDK остаются неизменными. Ваши существующие интеграции будут продолжать работать без каких-либо изменений.
