## Пример вывода

Вот код LaTeX для создания простой таблицы с тремя столбцами и двумя строками, включая строку заголовка с заголовками "Имя", "Возраст" и "Город", и строку данных со значениями "Джон", "25" и "Нью-Йорк":

```latex
\begin{table}[h]
\centering
\begin{tabular}{|c|c|c|}
\hline
Name & Age & City \\
\hline
John & 25 & New York \\
\hline
\end{tabular}
\caption{Sample Table}
\label{tab:sample}
\end{table}

```

Объяснение:

- Окружение `table` используется для создания плавающей таблицы, которая может быть автоматически позиционирована LaTeX.
- Необязательный параметр `[h]` предлагает LaTeX разместить таблицу "здесь" в потоке документа, если это возможно.
- Команда `\centering` центрирует таблицу горизонтально на странице.
- Окружение `tabular` используется для создания фактической структуры таблицы.
- Аргумент `{|c|c|c|}` определяет три центрированных столбца (`c`), разделенных вертикальными линиями (`|`).
- Команда `\hline` добавляет горизонтальные линии для разделения строки заголовка от строк данных и для обрамления таблицы.
- Символ `&` разделяет ячейки внутри строки, а `\\` отмечает конец строки.
- Команда `\caption` добавляет подпись к таблице, которая будет помечена как "Sample Table" в этом примере.
- Команда `\label` присваивает метку таблице, которая может использоваться для перекрестных ссылок в вашем документе.

Когда вы скомпилируете ваш документ LaTeX, этот код сгенерирует таблицу, которая выглядит следующим образом:

```
+---+---+---+
| Name | Age |   City   |
+---+---+---+
| John |  25 | New York |
+---+---+---+

```
