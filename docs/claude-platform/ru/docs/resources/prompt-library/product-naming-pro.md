# Профессиональное наименование продуктов

Создавайте запоминающиеся названия продуктов на основе описаний и ключевых слов.

---

> Скопируйте этот промпт в нашу [Консоль](/dashboard) разработчика, чтобы попробовать самостоятельно!

|        | Содержание |
| --- | --- |
| System | Ваша задача — генерировать креативные, запоминающиеся и продаваемые названия продуктов на основе предоставленного описания и ключевых слов. Названия продуктов должны быть краткими (2-4 слова), выразительными и легко понятными для целевой аудитории. Избегайте общих или слишком буквальных названий. Вместо этого стремитесь создать название, которое выделяется, передает суть продукта и оставляет неизгладимое впечатление. |
| User   | Описание: Беспроводные накладные наушники с шумоподавлением, 20-часовым временем работы от батареи и сенсорным управлением. Разработаны для аудиофилов и частых путешественников. Ключевые слова: погружающий, удобный, высококачественный, долговечный, удобный |

## Пример вывода

> 1. SoundOasis Elite
>    2.AudioZen Nomad
>    3.Serenity Pro
>    4.Tranquility Touch
>    5.Harmonix Journey
>    6.SonicSolace Roam
>    7.Auditory Bliss
>    8.Quietude Quest
>    9.Euphony Excursion
>    10.Acoustica Wanderlust
