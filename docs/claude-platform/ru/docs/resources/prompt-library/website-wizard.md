# Мастер веб-сайтов

Создавайте одностраничные веб-сайты на основе пользовательских спецификаций.

---

> Скопируйте этот промпт в нашу консоль разработчика [Console](/dashboard), чтобы попробовать его самостоятельно!

|        | Содержание |
| --- | --- |
| System | Ваша задача - создать одностраничный веб-сайт на основе заданных спецификаций, представленный в виде HTML-файла с встроенным JavaScript и CSS. Веб-сайт должен включать разнообразные привлекательные и интерактивные элементы дизайна, такие как выпадающие меню, динамический текст и контент, кликабельные кнопки и многое другое. Убедитесь, что дизайн визуально привлекателен, адаптивен и удобен для пользователя. HTML, CSS и JavaScript код должен быть хорошо структурирован, эффективно организован и правильно прокомментирован для читаемости и поддерживаемости. |
| User   | Создайте одностраничный веб-сайт для онлайн-платформы обучения под названием "EduQuest" со следующими функциями и разделами:<br/><br/>1. Фиксированная панель навигации со ссылками на категории курсов (Математика, Наука, Языки, Искусство) и строкой поиска.<br/>2. Героический раздел с видео-фоном, демонстрирующим студентов, обучающихся онлайн, динамическим слоганом, который меняется между "Учитесь в своем темпе", "Откройте новые увлечения" и "Расширьте свои горизонты" каждые 3 секунды, и кнопкой "Начать", ведущей к каталогу курсов.<br/>3. Раздел рекомендуемых курсов, отображающий карточки курсов с заполнителями для изображений курсов, названий, инструкторов и описаний.<br/>4. Интерактивный раздел "Пути обучения" с коротким тестом для определения стилей обучения и интересов, и кнопкой для начала теста.<br/>5. Раздел "Истории успеха" с отзывами довольных студентов, с заполнителями для текста отзывов и имен студентов.<br/>6. Подвал со ссылками на блог платформы, FAQ, политику конфиденциальности и кнопкой "Связаться с нами", которая открывает модальное окно с контактной формой и информацией службы поддержки клиентов.<br/><br/>Включите заполнители-наполнители для видео-фона, карточек курсов и отзывов. Встройте CSS-стили в тег `<style>` в разделе `<head>` и поместите JavaScript-код в тег `<script>` в конце раздела `<body>`.<br/><br/>JavaScript-код должен обрабатывать динамический слоган в героическом разделе, чередуя различные слоганы каждые 3 секунды. |

> Этот код создает одностраничный веб-сайт для "EduQuest" со следующими функциями и разделами:
> 
> Фиксированная панель навигации со ссылками на категории курсов (Математика, Наука, Языки, Искусство) и строкой поиска.
> 
> Героический раздел с заполнителем видео-фона, динамическим слоганом, который меняется между "Учитесь в своем темпе", "Откройте новые увлечения" и "Расширьте свои горизонты" каждые 3 секунды, и кнопкой "Начать".
> 
> Раздел рекомендуемых курсов, отображающий карточки курсов с заполнителями для изображений курсов, названий, инструкторов и описаний.
> 
> Раздел "Истории успеха" с отзывами довольных студентов, с заполнителями для текста отзывов и имен студентов.
> 
> Подвал со ссылками на блог платформы, FAQ, политику конфиденциальности и кнопкой "Связаться с нами".
> 
> CSS-стили встроены в тег `<style>` в разделе `<head>`, а JavaScript-код размещен в теге `<script>` в конце раздела `<body>`.
> 
> JavaScript-код обрабатывает динамический слоган в героическом разделе, чередуя различные слоганы каждые 3 секунды с помощью функции setInterval.
> 
> Обратите внимание, что видео-фон, изображения курсов и отзывы являются заполнителями и должны быть заменены на реальный контент.
