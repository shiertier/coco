# Server MCP remoti

Server MCP remoti disponibili per sviluppatori

---

Diverse aziende hanno distribuito server MCP remoti a cui gli sviluppatori possono connettersi tramite l'API del connettore MCP di Anthropic. Questi server espandono le capacità disponibili per sviluppatori e utenti finali fornendo accesso remoto a vari servizi e strumenti attraverso il protocollo MCP.

<Note>
    I server MCP remoti elencati di seguito sono servizi di terze parti progettati per funzionare con l'API Claude. Questi server
    non sono di proprietà, gestiti o approvati da Anthropic. Gli utenti dovrebbero connettersi solo a server MCP remoti di cui si fidano e
    dovrebbero rivedere le pratiche di sicurezza e i termini di ciascun server prima di connettersi.
</Note>

## Connessione ai server MCP remoti

Per connettersi a un server MCP remoto:

1. Rivedere la documentazione per il server specifico che si desidera utilizzare.
2. Assicurarsi di avere le credenziali di autenticazione necessarie.
3. Seguire le istruzioni di connessione specifiche del server fornite da ciascuna azienda.

Per ulteriori informazioni sull'utilizzo di server MCP remoti con l'API Claude, vedere la [documentazione del connettore MCP](/docs/it/agents-and-tools/mcp-connector).
