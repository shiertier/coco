# IPアドレス

Anthropicサービスは、インバウンド接続とアウトバウンド接続の両方に固定IPアドレスを使用します。これらのアドレスを使用して、Claude APIとコンソールへの安全なアクセスのためにファイアウォールルールを設定できます。これらのアドレスは予告なく変更されることはありません。

---

## インバウンドIPアドレス

これらは、Anthropicサービスが受信接続を受け取るIPアドレスです。

#### IPv4

`160.79.104.0/23`

#### IPv6

`2607:6bc0::/48`

## アウトバウンドIPアドレス

これらは、Anthropicがアウトバウンドリクエスト（例えば、MCPツール呼び出しを外部サーバーに行う場合）に使用する安定したIPアドレスです。
