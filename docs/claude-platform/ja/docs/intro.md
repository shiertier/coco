# Claudeの紹介

Claudeは、Anthropicによって構築された、高性能で信頼性が高く、インテリジェントなAIプラットフォームです。Claudeは、言語、推論、分析、コーディングなど、多くのタスクに優れています。

## 主な機能

Claudeは、テキスト、コード、画像を含む多くのタスクを支援できます。

<CardGroup cols={2}>
  <Card title="テキストとコード生成" icon="text-aa" href="/docs/ja/build-with-claude/text-generation">
