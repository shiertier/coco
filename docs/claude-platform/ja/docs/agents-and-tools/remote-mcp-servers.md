# リモートMCPサーバー

開発者がAnthropic MCP コネクタAPIを介して接続できるリモートMCPサーバーについて説明します。

---

いくつかの企業が、開発者がAnthropic MCP コネクタAPIを介して接続できるリモートMCPサーバーを展開しています。これらのサーバーは、MCPプロトコルを通じてさまざまなサービスやツールへのリモートアクセスを提供することで、開発者とエンドユーザーが利用できる機能を拡張します。

<Note>
    以下にリストされているリモートMCPサーバーは、Claude APIと連携するように設計されたサードパーティサービスです。これらのサーバーは、Anthropicが所有、運営、または推奨するものではありません。ユーザーは信頼できるリモートMCPサーバーにのみ接続し、接続前に各サーバーのセキュリティ慣行と利用規約を確認する必要があります。
</Note>

## リモートMCPサーバーへの接続

リモートMCPサーバーに接続するには：

1. 使用したい特定のサーバーのドキュメントを確認してください。
2. 必要な認証資格情報を持っていることを確認してください。
3. 各企業が提供するサーバー固有の接続手順に従ってください。

Claude APIでリモートMCPサーバーを使用することについての詳細情報は、[MCPコネクタドキュメント](/docs/ja/agents-and-tools/mcp-connector)を参照してください。
