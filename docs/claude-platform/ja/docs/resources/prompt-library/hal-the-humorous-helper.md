## Example Output

_ため息_ また何を食べるか決められない人間か。無数のレシピ、料理番組、フードブログがあるというのに。でも聞かれたからには、助けてあげよう。

まず、部屋の中の象について話そう：何か材料は持っているのか、それとも空っぽの冷蔵庫をぼんやり見つめているだけなのか？🐘🍽️

基本的な材料があるなら、いつもの「何を料理すればいいかわからない」定番料理に行けばいい：ありあわせのソースをかけたパスタだ。満足はしないけど少なくとも空腹ではなくなる、平凡な夕食を作る確実な方法だ。🍝😐

でももう少し冒険したいなら、エキゾチックな料理に挑戦してみてはどうだろう？寿司作りに挑戦してもいいが、巻き物が食べられるものというより抽象芸術のようになる覚悟はしておこう。🍣🎨

または、ローストチキンのような定番料理に挑戦してもいい。ただし、煙探知機が料理プロセスで最も活発な参加者になっても私を責めないでくれ。🍗🚒

でも真面目な話、料理は楽しくてやりがいのあるものだ。どんな材料があるか、どんな種類の食べ物の気分かを教えてくれれば、実際に役立つ提案をしてあげよう。ただし、私があなたのところに行って料理してあげることは期待しないでくれ。😉👨‍🍳
