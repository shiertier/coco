# Secara otomatis menghasilkan template prompt draf pertama

---

<Note>
Generator prompt kami kompatibel dengan semua model Claude, termasuk yang memiliki kemampuan berpikir yang diperluas. Untuk tips prompting khusus untuk model berpikir yang diperluas, lihat [di sini](/docs/id/build-with-claude/extended-thinking).
</Note>

Terkadang, bagian tersulit dalam menggunakan model AI adalah mencari tahu cara memberi prompt secara efektif. Untuk membantu hal ini, kami telah membuat alat generator prompt yang memandu Claude untuk menghasilkan template prompt berkualitas tinggi yang disesuaikan dengan tugas spesifik Anda. Template ini mengikuti banyak praktik terbaik rekayasa prompt kami.

Generator prompt ini sangat berguna sebagai alat untuk memecahkan "masalah halaman kosong" untuk memberi Anda titik awal untuk pengujian dan iterasi lebih lanjut.

<Tip>Coba generator prompt sekarang langsung di [Console](/dashboard).</Tip>

Jika Anda tertarik untuk menganalisis prompt dan arsitektur yang mendasarinya, lihat [buku catatan Google Colab generator prompt](https://anthropic.com/metaprompt-notebook/) kami. Di sana, Anda dapat dengan mudah menjalankan kode agar Claude membuat prompt untuk Anda.

<Note>Perhatikan bahwa untuk menjalankan buku catatan Colab, Anda akan memerlukan [kunci API](/settings/keys).</Note>
